# Nassau Logistics AI

A Streamlit logistics analytics project with sidebar pages, animated dropdown UI, route efficiency, geographic analysis, ship mode analysis, data quality checks, AI delay prediction, and executive summary.

## Run on Windows

1. Extract this zip file.
2. Open the extracted folder.
3. Double-click `run_project.bat`.
4. Wait for packages to install.
5. The app will open in your browser.

## Manual run

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## Important

Keep the folder structure unchanged:

```text
app.py
pages/
utils/
data/
requirements.txt
```

If `python` is not recognized, install Python 3.10+ and tick **Add Python to PATH** during installation.
